3. 字符串“open_door” 转换成 “OpenDoor” 、 ”make_by_id” 转换成 ”MakeById
```php
   $str="open_door";
   echo demo($str);
   function demo($str){
         $arr=explode("_", $str);
         foreach ($arr as $key=>$val){
               $arr[$key]=ucfirst($val);
         }
         return implode("", $arr);
   }
```
